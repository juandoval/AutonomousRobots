The aim of this task is to familiarise yourself with the principals of data encoding standards for transmission as a serial message. Reformatting information into a sequential stream of ‘bytes’ (encoding) followed by the splitting of the stream back into its components (decoding) is the basic principal that vast amounts of our civilisations technology is built upon. It underpins the internet with protocols such as TCP/IP, computing with protocols such as USB, and robotics with protocols such as ROS1 topics or MAVLink2 messages.
The task will be to interpret a serial message definition table and create a program to split an example set of data back into its constituent values. In other words, using a protocol definition, decode a serial data stream into human readable values.
